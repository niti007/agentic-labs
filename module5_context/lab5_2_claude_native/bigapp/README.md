# bigapp — a (pretend) large company project

Stand-in for a big, unfamiliar codebase you've just been dropped into. It has a few areas:

- `services/billing/` — invoices and billing config
- `services/auth/` — login and tokens
- `web/` — the website front end
- `shared/` — helpers used across the app

You don't need to understand the code. It's here so the AI has a real, multi-folder project
to *map* (Demo FF) and to *check* (Demo EE).
